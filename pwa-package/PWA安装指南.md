# 每日英语 - PWA安装指南

## 📱 什么是PWA？

PWA（Progressive Web App）是一种可以安装到手机上的网页应用，体验接近原生App，但不需要从应用商店下载。

## ✅ PWA的优势

- **完全离线使用** - 没有网络也能学习
- **像App一样安装** - 在手机主屏幕有图标
- **节省空间** - 比原生App小很多
- **自动更新** - 打开即用，无需手动更新
- **跨平台** - iOS和Android都能用

## 📲 安装方法

### 📱 Android手机（推荐）

1. **打开网页**
   - 使用Chrome浏览器访问应用
   - 确保网络连接正常（首次安装需要）

2. **添加到主屏幕**
   - 点击浏览器右上角菜单（三个点）
   - 选择"添加到主屏幕"或"安装应用"
   - 点击"添加"或"安装"

3. **完成**
   - 返回主屏幕，看到"每日英语"图标
   - 点击即可使用

### 🍎 iPhone/iPad

1. **打开网页**
   - 使用Safari浏览器访问应用
   - 确保网络连接正常

2. **添加到主屏幕**
   - 点击底部分享按钮（方框加向上箭头）
   - 选择"添加到主屏幕"
   - 点击右上角"添加"

3. **完成**
   - 返回主屏幕，看到"每日英语"图标
   - 点击即可使用

### 💻 电脑使用

- 直接在浏览器打开 index.html 即可
- 或使用本地服务器：
  ```bash
  # Python
  python -m http.server 8000
  
  # 访问 http://localhost:8000
  ```

## 🎯 使用建议

### 首次使用
1. 安装到手机后，先联网打开一次（让PWA缓存所有文件）
2. 之后就可以离线使用了

### 日常学习
- 每天打开应用学习15-20分钟
- 系统会自动保存学习进度
- 不需要联网也能使用

### 数据备份（可选）
- 学习进度保存在浏览器中
- 如需备份，可使用浏览器同步功能

## ⚠️ 注意事项

1. **图标显示**
   - 如果图标没有显示，确保手机有网络
   - 清除浏览器缓存后重新添加

2. **离线使用**
   - 首次安装后，必须联网打开至少一次
   - 之后才能离线使用

3. **更新应用**
   - 只需在有网络时打开应用
   - 会自动更新到最新版本

4. **删除应用**
   - Android：长按图标 → 卸载
   - iPhone：长按图标 → 删除应用

## 🆘 常见问题

### Q: 为什么图标没有显示？
A: 确保使用了正确的浏览器（Android用Chrome，iPhone用Safari），并且有网络连接。

### Q: 安装后需要联网吗？
A: 首次使用需要联网缓存文件，之后可以完全离线使用。

### Q: 如何更新应用？
A: 只要有网络时打开应用，会自动更新，无需手动操作。

### Q: 学习进度会丢失吗？
A: 不会。进度保存在浏览器中，只要不清除浏览器缓存就不会丢失。

### Q: 可以在电脑和手机同步进度吗？
A: 当前版本是独立的，如需同步可以：
- 使用同一账号的浏览器同步功能
- 或者导出学习数据到另一设备
